# 🧠 Personality Test Bot – OCEAN Model

This is a chatbot-based personality testing system built on the **OCEAN personality model**. The system presents interactive questions, analyzes user responses, and generates a personalized personality profile in real-time.

---

## 📖 About the Project

The goal of this project is to provide a simple, engaging, and immediate way for users to understand their personality traits. Using the **OCEAN (Big Five)** model — Openness, Conscientiousness, Extraversion, Agreeableness, and Neuroticism — the chatbot evaluates responses and generates a result without requiring expert supervision.

---
## ✨ Features

 -🔹 Chatbot-based personality test
 -🔹 Uses the OCEAN (Big Five) model
 -🔹 Likert-scale-based question responses
 -🔹 Real-time result generation
 -🔹 Simple and clean user interface

## Setup

1. Create a virtual environment (recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and navigate to `http://localhost:5000`

## Usage

1. Click "Start Test" to begin
2. Answer each question by selecting a number from 1 to 5
   - 1: Strongly Disagree
   - 2: Disagree
   - 3: Neutral
   - 4: Agree
   - 5: Strongly Agree
3. Click "Submit Answers" to see your results
4. Review your personality trait scores and descriptions
